position: static;

По умолчанию все элементы имеют position: static. Это значение явно указывать не обязательно. Такие элементы находятся в потоке и занимают определенное пространство в документе. Свойства top: ...; left: ...; righ: ...; bottom: ...; никак не влияют на такой элемент

p1 - position: static;
c1 - position: absolute;

position: relative;

Элементы с position: relative; остаются в потоке и занимают определенное пространство в документе. Могут изменять свое положение относительно первоначальной позиции с помощью свойств top: ...; left: ...; righ: ...; bottom: ...; Их используют для размещения маленьких объектов относительно других на странице.

p2 - position: relative; top: 20px; left: 50px;
c2 - position: absolute;

position: absolute;

Элементы с position: absolute выпадают из потока и располагаются в заданной позиции относительно ближайшего родителя с position: relative | absolute | fixed. Если такого родителя нету, то элемент позиционируется относительно body

p3 - position: absolute; right: 20px; left: 50px;
c3 - position: absolute;

position: fixed;

Элементы с position: fixed; выпадают из потока и фиксируют свое положение относительно экрана (окна браузера). Управлять их положением можно с помощью свойств top: ...; left: ...; righ: ...; bottom: ...;

p4 - position: fixed; bottom: 20px; left: 30px;
c4 - position: absolute;
